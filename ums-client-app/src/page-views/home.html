InterNations

User management Application

Front End: ES6/JavaScript only, no external libraries/framework used. No NPM packages used as dependencies.
  • Created Custom Web components.
  • Created own Custom HTML template rendering service
  • Created own Custom Internationalization and language service
  • Created own Router service
  • Created own Loader service, that shows loaders across application components.
  • Application Theming implemented using CSS variables
  • ES Linting
  • Webpack: Build and Bundling
Back End: Node & Express js
Database: json-server (No-SQL)
